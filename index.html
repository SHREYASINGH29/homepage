Selected Projects:
Modeling Implementation and Optimization in LLVM

  • Automatically generates abstract representation of LLVM IR
  • Uses textual description of the modeling as input
  • Eliminates the need of finding manual patterns in the IR by conservatively replacing uses with defs for the LLVM generated temporaries
  • Optimizes the representation to match source level constructs

Live Variable Analysis in LLVM

  • Implemented intra-procedural live variable analysis in LLVM
  • Upgraded to an inter-procedural version

Strong Live Variable Analysis in LLVM

  • Implemented intra-procedural and inter-procedural strong live variable analysis in LLVM

Modelling LLVM IR instructions

  • Modelled LLVM IR instructions into a small grammar close to C
  • Handled memory instructions, Load, Store, Alloca, BitCast, and GetElementPointer
  • Binary arithmetic operations, Add, Mul, Sub, and UDiv
